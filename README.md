## :point_down:Steps to initialize the project:

- Clone the repository

$ git clone https://github.com/anjupriya-v/bmi-calculator.git

- Redirect to the cloned repo directory

- update the npm

`npm update --force`

- Install the dependencies

`npm install`

- Start the project 

`nodemon index.js`
